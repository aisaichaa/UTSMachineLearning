Clustering

2. Gambaran Singkat Proyek
Tahapan yang dilakukan meliputi:
- IEDA
- Preprocessing (median fill dan clipping outliers)
- Scaling
- Menentukan jumlah cluster menggunakan Elbow Method dan Silhouette Score

- Analisis dan interpretasi cluster
- Visualisasi 2D menggunakan PCA
3. Model dan Hasil Metrik
- Model yang digunakan: K-Means
- Metrik evaluasi: Silhouette Score dan analisis centroid

Fraud Detection

1. Tujuan Repositori: Repositori ini bertujuan untuk membangun pipeline Machine Learning end-to-end untuk mendeteksi transaksi fraud menggunakan dataset transaksi dan identity.
2. Gambaran Singkat Proyek
Proyek ini mencakup:
- Penggabungan dataset transaksi dan identity
- IEDA
- Preprocessing 
- Penanganan imbalance dengan undersampling
- Training model LightGBM
- Hyperparameter tuning
- Evaluasi menggunakan ROC-AUC, PR-AUC, classification report, dan confusion matrix
